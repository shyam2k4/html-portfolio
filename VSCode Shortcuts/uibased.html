UI Based Shortcuts

  • Ctrl + B - Toggle Sidebar

  • Ctrl + ` - Toggle Terminal

  • Ctrl + P - Quick Open Menu
    (To Search For Specific File)

  • Ctrl + P, and then Enter "@" - Quick Menu
    (To Search For Specific Element/Tag in a File)

  • Ctrl + Up/Down - Scroll

  • Ctrl + \ - Split Editor

  • Ctrl + W - Close Tab
